# Desktop downloads

Download the installer that matches your device from the [latest GitHub Release](https://github.com/luobuchao0321/wechat-article-editor/releases/latest).

| System | Recommended file |
| --- | --- |
| macOS Apple Silicon | `ContentCraft-<version>-arm64.dmg` |
| macOS Intel | `ContentCraft-<version>-x64.dmg` |
| Windows 64-bit | `ContentCraft-Setup-<version>-x64.exe` |
| Windows 32-bit | `ContentCraft-Setup-<version>-ia32.exe` |
| Linux | `ContentCraft-<version>-x86_64.AppImage` or `.deb` |

Each release includes SHA256 checksum files. Compare the checksum after download when distributing the app inside a team.

## macOS and Windows prompts

Until code-signing and notarization are enabled, macOS Gatekeeper or Windows SmartScreen may show a warning for a newly released build. Download only from the official GitHub Release page and verify the checksum. Signed builds are planned for a future release.
